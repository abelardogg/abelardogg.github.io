# Project Rules

## Tech stack
- Vite + React + TypeScript (SPA estática)
- Tailwind CSS v4 (estilos)
- Framer Motion (animaciones)
- Lucide React (íconos)
- GitHub Pages deployment vía GitHub Actions

## Code style
- No comments in code unless strictly necessary
- Functional components con hooks
- camelCase variables, const/let only
- Mobile-first responsive design
- Sin dependencias externas pesadas

## Conventions
- Componentes en `src/components/`, datos en `src/data/`
- Assets estáticos en `public/assets/`
- Tailwind utility classes sobre CSS custom
- Imágenes de preview en `.webp` con `loading="lazy"`

## Git
- No commits unless explicitly requested
- Keep commit messages in Spanish or English (match existing)
