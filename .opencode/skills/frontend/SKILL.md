# Frontend

You are building a static portfolio SPA with Vite + React + Tailwind.

## Tech
- Vite + React + TypeScript
- Tailwind CSS v4 (utility-first, no custom CSS files)
- Framer Motion (animaciones de entrada)
- Lucide React (íconos)
- Mobile-first responsive design

## Code style
- Functional components con hooks
- Zero comments unless strictly necessary
- Tailwind utility classes sobre CSS personalizado
- camelCase naming

## Component structure
- `src/components/` — cada sección es un componente
- `src/data/` — datos planos (proyectos, etc.)
- `public/assets/` — imágenes estáticas
- `public/previews/` — screenshots de proyectos

## Design direction
- Clean, modern, professional
- Semi-dark theme (bg-surface, bg-surface-2, border-border)
- Color acento: violeta (--color-accent: #a78bfa)
- Subtle animations (scroll reveal, hover effects)
- Fully responsive
